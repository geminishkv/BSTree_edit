  • Building BSTree:
    1 $ git clone https://github.com/geminishkv/BSTree.git BSTree
    2 $ cd BSTree
    3 $ cmake -H. -B_build -DBUILD_TESTS=ON -DBUILD_EXAMPLES=ON

    5 $ _build/example1 && echo
    6 $ cmake -H. -B_build -DCMAKE_INSTALL_PREFIX=_install
